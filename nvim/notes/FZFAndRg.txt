:FZF
----
- Search for Files

:Files
------
- Search for Files in current project (based on .git)
  - Shows File Preview

:Rg
---
- This is ripgrep
- looks inside of files for strings

:BLines
-------
- Searches for lines in current buffer containing a string

:Lines
------
- Searches for lines in all buffers containing a string

:History:
---------
- Will pop open a window with all of the previous commands you've ran
